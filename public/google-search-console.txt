# Google Search Console 验证文件
# 部署后将此内容替换为 Google 提供的 HTML 验证文件内容
# 或使用 DNS/meta 标签验证方式

# 操作步骤：
# 1. 登录 https://search.google.com/search-console
# 2. 添加资源 → 选择"网址前缀"，输入你的域名
# 3. 选择"HTML文件"验证方式
# 4. 下载 google*.html 文件，放到 /public/ 目录下
# 5. 重新构建部署
# 6. 点击"验证"按钮完成验证
# 7. 验证成功后在 Search Console 提交 sitemap: https://你的域名/sitemap.xml
